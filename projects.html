Projects

Discover more about the research currently being done at VHLab!

Cultural Crowds (2015 - 2019) Complete

Soraia Raupp Musse Rodolfo Favaretto

This project aims to investigate cultural aspects in crowds in video sequences. It´s supported by ONRG Office of Naval Research Global.

HealthCrowds (2015 - 2016) Complete

Soraia Raupp Musse Norman Badler

This project aims to investigate the reaction of heterogeneous crowds in panic situations. Supported by CAPES - Estágio Senior Developed at UPENN

Crowd Modeling and Simulation (2013 - 2017) Complete

Soraia Raupp Musse Estevão testa Gabriel Lima Cliceres dal Bianco Amyr Fontes Neto

This project aims to develop new methods in crowd simulation, using as basis the two softwares developed at VHLAB: CrowdSim and BioCrowds.

Projeto FACES (2013 - 2017) Complete

Soraia Raupp Musse Adriana Braun Rossana Queiroz

This project aims to investigate performance driven animation and facial recognition in computer animation. Supported by CNPq - BOLSA PQ1

Identidade (2013 - 2016) Complete

Soraia Musse Leandro Dihl Marlon Fontoura

This project aims to identify people in video sequences based on their cloths and/or grouping membership. The main goal is to avoid the facial identification.
